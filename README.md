AZBA3D was vibecoded with the help of Claude and ChatGPT. 

A custom Python script was used to process .nii.gz and text files from _A 3D Adult Zebrafish Brain Atlas (AZBA) for the Digital Age_ (Kenney et. al, 2021) into lightweight .glb files. These .glb files can be viewed and interacted with at **https://ianfences.github.io/AZBA3D/**

The original dataset can be downloaded at **https://datadryad.org/dataset/doi:10.5061/dryad.dfn2z351g**

**Controls:**

Left-drag to rotate. Right-drag to pan. Scroll to zoom. Shift-click to select multiple regions.
