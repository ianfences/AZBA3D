AZBA3D was vibecoded with the help of Claude and ChatGPT. 

AZBA3D took .nii.gz and text files from _A 3D Adult Zebrafish Brain Atlas (AZBA) for the Digital Age_ (Kenney et. al, 2021) and proccessed them into lightweight .glb files using a custom Python script. These lightweight files are easily viewed on the HTML viewer. 

The original dataset can be downloaded at **https://datadryad.org/dataset/doi:10.5061/dryad.dfn2z351g**

**Controls:**

Left-drag to rotate. Right-drag to pan. Scroll to zoom. Shift-click to select multiple regions.
